# Digest Schema

Digests summarize related raw intelligence entries.

## Location

```text
intelligence/digests/YYYY-MM-DD-<topic>.md
```

## Frontmatter

```yaml
created_at: 2026-05-29T12:00:00+08:00
type: intelligence_digest
topic: ""
sources: []
tags: []
```

## Body

```markdown
# Summary

# Signals

# Pattern

# Next Actions
```
